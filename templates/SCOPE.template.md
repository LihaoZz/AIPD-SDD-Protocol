# Scope

## Product Surface

- what the user can do
- what the system must do

## Version Target

- what is included now
- what is intentionally deferred

## Acceptance Definition

- what must be true for this scope to count as successful

## Out Of Scope

- items explicitly not included in this iteration

---

## 中文翻译

### 范围

#### 产品表面

- 用户可以做什么
- 系统必须做什么

#### 版本目标

- 当前版本包含什么
- 当前版本故意延期什么

#### 验收定义

- 要把这个范围算作成功，必须满足什么条件

#### 范围外

- 本轮明确不做的内容
