# Project Constitution

## Product Goal

What problem this project solves and for whom.

## Non-Negotiables

- constraints that must not be violated

## Current Architecture Direction

- chosen stack
- major boundaries
- important limitations

## Quality Gates

- required tests
- security expectations
- performance expectations if relevant

## Approval Rules

- what the Builder can decide alone
- what requires spec approval

---

## 中文翻译

### 项目宪章

#### 产品目标

这个项目为谁解决什么问题。

#### 不可谈判项

- 绝对不能违反的约束

#### 当前架构方向

- 选定技术栈
- 主要边界
- 重要限制

#### 质量门槛

- 必需测试
- 安全要求
- 如果相关，还包括性能要求

#### 审批规则

- Builder 可以自行决定什么
- 哪些内容必须经过规格审批
