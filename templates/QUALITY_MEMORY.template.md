## 中文翻译

### 质量记忆模板

#### 目的

只记录跨 function blocks 和 mission blocks 可复用的质量经验。

#### 稳定规则

- `rule`：稳定规则
- `why`：原因
- `source_refs`：来源引用

#### 重复失败模式

- `pattern`：失败模式
- `seen_in`：出现位置
- `avoid_next_time`：下次避免方式

#### 已验证有效的做法

- `pattern`：有效模式
- `use_when`：适用场景
- `evidence_refs`：证据引用

#### 高风险区域

- `area`：区域
- `common_failure`：常见失败
- `required_checks`：必需检查

#### 最近经验

- `date`：日期
- `lesson`：经验
- `source_mb`：来源 MB

#### 使用规则

- 不要把这个文件写成记录每个 MB 的流水账。
- 只记录可复用经验、重复失败、稳定有效做法和高风险提醒。
- 只有在以下情况才更新：
  出现重复失败、
  确认了新的高价值经验、
  或某条规则应沉淀为稳定规则。
- 普通成功且没有可复用洞见时，不要写入。
- 每条经验尽可能回指到具体的 `fb` 或 `mb`。
